Data Loading: Listing data is read from a CSV file.
Data Cleaning: Missing, incorrect, or inconsistent data is identified and corrected.
Data Transformation: Data is converted into a format suitable for analysis, and new features are created.
Hive Integration: The transformed data is stored in Hive for future analysis.
Technologies Used:

Spark
Hive
Python
Project Goal:

This project aims to demonstrate the power of Spark for large-scale data processing and analysis. It also provides a clean and organized dataset that can serve as a foundation for data analysis projects related to the second-hand car market.

Note: The utility.my_helpers module in the project provides customizable functions for data cleaning and transformation steps. These functions can be adapted to different datasets or analysis needs.
